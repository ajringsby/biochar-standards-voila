# Biochar standards crosswalk (Voila + Binder)

This repo publishes a Voila dashboard version of `app.ipynb`.

## Launch (Binder + Voila)

Launch link
https://mybinder.org/v2/gh/ajringsby/biochar-standards-voila/HEAD?urlpath=voila/render/app.ipynb

Notes:
- First launch may take a bit (Binder builds the environment).
- Each user gets their own temporary session.

## Local run

```bash
pip install -r requirements.txt
voila app.ipynb
```
